# scrumpkr.net

Minimalist, no-login Planning Poker for agile teams. Create a room, share the link, start estimating.

Built for teams who want to get straight to the point — no accounts, no setup, no friction.

## Features

- **No login required** — jump in with just your name
- **Persistent rooms** — rooms stay alive so you can come back later
- **Real-time voting** — see votes live as they come in
- **Multiple estimation decks** — choose the scale that fits your team
- **Spectator mode** — observers can watch without voting
- **Vote history** — review past rounds without losing context
- **Privacy first** — no trackers, no personal data collection, no ads

## Tech Stack

- [Cloudflare Workers](https://workers.cloudflare.com/) + [Durable Objects](https://developers.cloudflare.com/durable-objects/)
- [React 19](https://react.dev/) + [React Router 7](https://reactrouter.com/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
